Database Management System (DBMS)

Overview
This repository contains Database Management System (DBMS) concepts, SQL queries, schemas, and examples designed to help students and beginners understand how databases work in real-world applications.
The project focuses on relational databases, SQL operations, and core DBMS principles with practical implementations.

Objectives
Understand fundamental DBMS concepts
Learn SQL for data definition and manipulation
Practice database design using ER models
Implement normalization and constraints
Perform CRUD operations efficiently
